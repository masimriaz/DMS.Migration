# Data Migration Platform - Logo Assets

## Brand Identity

**Product Name:** Data Migration Platform  
**Tagline:** Enterprise Platform  
**Brand Colors:**

- Primary Blue: `#3B82F6`
- Dark Blue: `#2563EB`
- Light Blue: `#60A5FA`
- Dark Text: `#1E293B`
- Medium Text: `#64748B`

## Logo Files

### 1. **logo.svg** (200x60px)

- **Usage:** Full horizontal logo with text
- **Where:** Email signatures, documents, presentations
- **Contains:** Icon + "Data Migration" text + tagline

### 2. **logo-icon.svg** (44x44px)

- **Usage:** Sidebar/navigation icon
- **Where:** Application sidebar, small UI elements
- **Contains:** Icon only with subtle background

### 3. **favicon.svg** (32x32px)

- **Usage:** Browser tab icon
- **Where:** Browser favicon, app icons
- **Contains:** Simplified icon optimized for small sizes

## Design Elements

### Icon Symbolism

The logo features a modern, abstract representation of data migration:

- **Two Servers/Databases:** Representing source and destination systems
- **Arrow:** Symbolizing data transfer/migration
- **Data Dots:** Representing individual data items being migrated
- **Gradient:** Modern, professional aesthetic

### Typography

- **Primary Font:** Inter, Segoe UI, System UI (sans-serif)
- **Logo Weight:** 700 (Bold) for "Data Migration"
- **Tagline Weight:** 500 (Medium) for "ENTERPRISE PLATFORM"

## Usage Guidelines

### ✅ DO:

- Use the full logo on light backgrounds
- Maintain minimum clear space around logo (equal to height of icon)
- Use provided SVG files for scalability
- Keep original aspect ratios

### ❌ DON'T:

- Distort or stretch the logo
- Change the brand colors
- Add effects (shadows, outlines, etc.)
- Use on busy backgrounds without sufficient contrast

## Accessibility

- All logos use semantic SVG markup
- Color contrast ratios meet WCAG AA standards
- SVG format ensures crisp rendering at any size
